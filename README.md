## Como Rodar Localmente


1. Install dependency:
```sh
   npm install
```
2. Build project:
```sh
  npm run build
```
3. Start the development server:
```sh
  npm run dev
```
4. Open the project in your browser at `http://localhost:5173`.

## Guideline:
  1. 3x3 grid
  2. Name or ID search
  3. Direct API data with over 1000 pokemon
  4. Dark mode always on
